## ‚ú® About The Project

**Valentine Quest** is a fun, interactive, retro-themed digital experience designed to pop the big question in style. Built with the latest tech stack providing a nostalgic Windows 95/98 aesthetic, it takes the user on a playful journey through various "security checks" before delivering the final message.

It's not just a website; it's a **quest for love**.

## üöÄ Features

- **Retro Aesthetic**: Custom-styled windows, buttons, and layouts mimicking classic OS interfaces.
- **Runaway "No" Button**: The "No" button mischievously evades the cursor, making "Yes" the only option.
- **Interactive "Security" Checks**:
  - A quirky **Captcha** system (because love is serious business).
  - A **Fake Loading Screen** that generates humorous status updates like "Negotiating with Fate...".
- **Dynamic Desktop**: Clickable desktop icons like "Love_Letter.txt" and "Recycle Bin".
- **Mobile Responsive**: Fully optimized for both desktop and mobile declarations of affection.
- **Celebration Mode**: Confetti and a heartfelt success screen upon completion.

## üíª Getting Started

Want to run this locally? Follow these simple steps:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/valentine-quest.git
    cd valentine-quest
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    # or
    bun install
    ```

3.  **Run the development server:**

    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) to see the magic happen.
